# 📰 Fake News Generator

A fun and powerful Fake News Generator Web App built using Python and Streamlit.  
This app generates unlimited unique fake news headlines using random data and CSV datasets.

---

## 🚀 Features

- 🔥 Unlimited Fake News Generation
- 🎲 Random + CSV Data Mode
- 🧠 Smart Sentence Templates
- 📁 CSV Dataset Support
- 💾 Saves Generated News to File
- 🌐 Web App using Streamlit

---

## 🛠️ Tech Stack

- Python 🐍
- Streamlit 🌐
- CSV 📄
